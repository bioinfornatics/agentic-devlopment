# Product Vision

Version: 1.0.0 | Status: Active

## Problem Statement

Building with AI agents today means no memory, no durable tasks, no quality gates.
Agents "vibe-code" — fast but unreliable for production.

## Vision

A portable, composable operating system for agentic software development.

## Target Users

| Persona | Pain | Solution |
|---|---|---|
| Solo developer | Agents forget context | SDD loop + Beads |
| Tech lead | Hard to enforce standards | Skills + recipes |
| Product team | Features get lost | /discover + /spec + KG |
| Contributor | Hard to improve quality | A/B eval suite |

## Success Metrics

- Skill delta > 0 on all skills
- R1:ac-test-gap = 0
- Recipe validation 100%
- Tests 32+ passing

## Principles

1. Composable — each component works standalone or together
2. Spec-anchored — specs persist, code follows specs
3. Self-improving — A/B evals and KG gaps drive improvement
4. Minimal blast radius — scoped, claimed, verified
5. No sudo — always user-space alternatives
