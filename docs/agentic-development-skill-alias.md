# Agentic development skill alias policy

Preferred human spelling: agentic-development.

Compatibility identifier: agentic-devlopment.

The misspelled identifier is retained on disk for current recipes, evals, Beads history, and generated docs. The skill frontmatter declares agentic-development as an alias/preferred spelling. A future structural rename must update recipes, eval JSON, generated tables, specs, and the KG pipeline in one SDD cycle.
