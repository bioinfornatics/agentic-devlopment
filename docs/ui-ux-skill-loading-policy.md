# UI/UX skill loading boundaries

The UI/UX skill set is layered: agentic-ux core UX workflow; ui-quality visual checks; ux-quality journeys; atomic-design hierarchy; design-systems-arch tokens and governance; cognitive-ux heuristics; design-critique-case-studies optional critique references; frontend-blueprint implementation handoff. Agents and recipes load the smallest required set first, then deep-load optional skills only when request or evidence triggers them.
